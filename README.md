# 📊 Sales Dashboard & Regional Analysis | Power BI Project

## 📌 Project Overview
This project presents an interactive **Sales Dashboard** built using Power BI, combined with exploratory data analysis (EDA) to uncover key business insights.  

The goal is to analyze regional sales performance, identify trends, and support data-driven decision-making.

---

## 🎯 Objectives
- Analyze sales performance across regions  
- Identify top-performing products and categories  
- Track revenue and profit trends  
- Understand customer and regional behavior  
- Build an interactive dashboard for visualization  

---

## 📊 Dataset Description
The dataset includes sales-related information such as:

- Order Date  
- Region  
- Product Category & Sub-Category  
- Sales Revenue  
- Profit  
- Customer Segment  
- Quantity Sold  

---

## 🔍 Business Problems Solved

### 📍 Regional Analysis
- Compare sales performance across regions  
- Identify high and low-performing regions  

### 📦 Product Analysis
- Determine best-selling products and categories  
- Analyze sub-category performance  

### 💰 Sales & Profit Analysis
- Track revenue and profit trends over time  
- Identify profitable vs loss-making areas  

### 👥 Customer Analysis
- Analyze customer segments  
- Understand buying patterns  

### 📈 Trend Analysis
- Monthly and yearly sales trends  
- Growth and decline patterns  

---

## 📊 Dashboard Features
- 📌 Interactive filters (Region, Category, Time)  
- 📈 Sales & Profit trend charts  
- 📊 Category-wise performance visuals  
- 🗺️ Regional comparison analysis  
- 📉 Profitability insights
